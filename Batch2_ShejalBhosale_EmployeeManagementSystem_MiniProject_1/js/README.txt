--------------------------------------------------
PROJECT OVERVIEW
--------------------------------------------------
This is a browser-based Employee Management System built using HTML, CSS, and JavaScript. 
The application allows admins to manage employee records efficiently without using any backend or database.

All data is stored in JavaScript arrays and managed dynamically through the UI.

--------------------------------------------------
FEATURES
--------------------------------------------------
- Admin Signup & Login
- Add Employee
- Edit Employee Details
- Delete Employee
- View Employee List
- Search Employees
- Filter by Department & Status
- Sort Employees
- Dashboard Summary (Total, Active, Inactive)
- Form Validation
- Unit Testing using Jest

--------------------------------------------------
HOW TO RUN THE APPLICATION
--------------------------------------------------
1. Download and unzip the project folder
2. Open the folder
3. Double-click on index.html
   OR
   Right-click → Open with Browser

The application will run directly in the browser.

--------------------------------------------------
HOW TO RUN TESTS
--------------------------------------------------
1. Open terminal in project root folder
2. Install dependencies:

   npm install

3. Run tests:

   npm test

All test cases will execute using Jest.
